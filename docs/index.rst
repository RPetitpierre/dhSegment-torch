dhSegment : Generic framework for historical document processing
================================================================

**dhSegment** is a tool for Historical Document Processing. Its generic approach allows to segment regions and
extract content from different type of documents.

This the documentation of the PyTorch version of **dhSegment** which the one currently in development.
The no longer developed Tensorflow version can still be found at this address: `<https://dhsegment.readthedocs.io>`_.
